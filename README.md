## Featured Projects

### 1. Agentic RAG System for Reliable Physics QA

A physics question-answering assistant built using **LangGraph, ChromaDB, Groq API, LLaMA 3.3 70B, Streamlit, and RAGAS**.

The project focuses on grounded answering, retrieval quality, memory, tool usage, and evaluation instead of simple prompt-based responses.

**What I built:**

- Built a RAG-based QA system over structured physics documents
- Used LangGraph to manage routing, retrieval, memory, and response generation
- Used ChromaDB for semantic retrieval
- Integrated a calculator tool for numerical queries
- Added fallback handling for low-context and out-of-scope questions
- Tested the system with concept questions, formula queries, memory checks, and red-team cases
- Evaluated outputs using RAGAS

**Evaluation results:**

| Metric | Score |
|---|---:|
| Faithfulness | 0.82 |
| Answer Relevance | 0.87 |
| Context Precision | 0.78 |

**Tech:** Python, LangGraph, ChromaDB, Groq API, LLaMA 3.3 70B, Sentence Transformers, Streamlit, RAGAS
